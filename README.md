# Project

This repository contains the data and analysis code for my final paper for LAW7517: Choosing Among Policy Instruments (Winter Quarter 2025/26).

It contains two subdirectories:

- `analysis`: This directory the .Rmd notebook for generating plots.
- `plots`: This directory contains the plots generated by the code in `analysis`.
- `data`: This directory contains the data imported into R by the code in the .Rmd notebooks in `analysis`.
